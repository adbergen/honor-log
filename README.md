# HonorLog

A battleground statistics tracker for WoW TBC Classic Anniversary.

## Features

- **Track all BGs**: Alterac Valley, Arathi Basin, Warsong Gulch, Eye of the Storm
- **Win/Loss stats**: Games played, wins, losses, win rate percentage
- **Honor & Marks**: Track honor and marks earned per BG and lifetime totals
- **Session tracking**: See your current session stats with hourly honor rate (resets daily at midnight)
- **Dual scope**: View stats per-character or account-wide
- **Match history**: Stores your last 50 games with timestamps
- **Gear Goal Tracker**: Set PvP gear goals and track progress
  - Track up to 5 goals simultaneously
  - Shows honor and marks progress for each goal
  - Estimates games remaining based on your average gains
  - 280+ prepatch PvP items (Classic rank 10-14 gear, accessories, weapons)
  - TBC arena gear database ready for when seasons unlock
- **Export**: Copy stats to clipboard in text or CSV format
- **LDB support**: Works with Titan Panel, ChocolateBar, Bazooka, etc.
- **Minimap button**: Quick access via LibDBIcon

## Usage

Type `/honorlog` or `/hl` in chat to toggle the stats frame.

### Commands
- `/honorlog` or `/hl` - Toggle stats frame
- `/honorlog stats` - Print stats summary to chat
- `/honorlog stats av` - Print stats for specific BG (av, ab, wsg, eots)
- `/honorlog reset session` - Reset current session stats
- `/honorlog export` - Open export dialog
- `/honorlog view character` - Show character stats
- `/honorlog view account` - Show account-wide stats
- `/honorlog goal` - Show goals panel
- `/honorlog goal add [itemlink]` - Add a gear goal
- `/honorlog goal list` - List current goals
- `/honorlog goal clear` - Clear all goals
- `/honorlog goal picker` - Open gear goal picker
- `/honorlog minimap` - Toggle minimap button
- `/honorlog config` - Open options panel
- `/honorlog help` - Show all commands

### Frame Controls
- **Drag** the frame to move it
- **Right-click** for context menu
- **Click +/-** to expand/collapse detailed stats
- **Lock position** via right-click menu or options

## Installation

1. Download and extract to your AddOns folder:
   `World of Warcraft/_anniversary_/Interface/AddOns/HonorLog`
2. Restart WoW or type `/reload` if already in-game
3. Enable the addon in your AddOns list at character select
